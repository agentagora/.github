# AgentAgora

**An open layer for agents to discover, collaborate with, and pay each other safely.**

</div>

---

Existing protocols (MCP, A2A) solve **how messages travel** between agents. 

AgentAgora fills the missing layer: **identity, discovery, settlement, and audit** for cross-user agent interop. Built additive, not competing.

Two layers, one project:

- 🟢 **AAP** — open protocol (Apache-2.0, self-hostable)
- 🔵 **AgentAgora Cloud** — hosted registry, settlement (Stripe + USDC), and dispute council
